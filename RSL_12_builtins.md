# Встроенные процедуры RSL

> RSL (R-Style Language) — Руководство программиста

## Общие сведения

В данном разделе представлен список всех стандартных процедур RSL:

- Типы и значения переменных
- Работа со строками
- Параметры процедур
- Математические функции
- Работа с файлами и структурами
- Управление файлами и каталогами
- Обработка ошибок и прочие

**Соглашения:**
- В квадратных скобках `[...]` — необязательные параметры
- Как правило, процедуры возвращают `TRUE` при успехе, `FALSE` при неудаче

## Процедуры ввода данных с клавиатуры

Процедуры присваивают переменной (первый параметр) значение, введённое пользователем. Предыдущее значение переменной используется как значение по умолчанию.

| Параметр | Тип | Описание |
|----------|-----|----------|
| `id` | Выходной | Переменная для сохранения введённого значения |
| `prompt` | `V_STRING` | Текст приглашения (по умолчанию: `"Введите значение: "`) |
| `len` | `V_INTEGER` | Ширина поля ввода |
| `hide` | `V_BOOL` | Если `TRUE` — ввод отображается звёздочками |
| `pos` | `V_INTEGER` | Количество знаков после десятичной точки |

**Возвращаемое значение:** `TRUE` при успехе, `FALSE` если пользователь ничего не ввёл (переменная не изменяется).

### GetInt

```rsl
GetInt ( id [, prompt, len [, hide ] ] )
```

Присваивает переменной типа `V_INTEGER` введённое значение. Ширина поля по умолчанию: **12 символов**.

### GetDouble

```rsl
GetDouble ( id [, prompt, len [, hide [, pos ] ] ] )
```

Присваивает переменной типа `V_DOUBLE` введённое значение. Ширина поля по умолчанию: **24 символа**.

### GetMoney

```rsl
GetMoney ( id [, prompt, len [, hide ] ] )
```

Присваивает переменной типа `V_MONEY` введённое значение. Ширина поля по умолчанию: **18 символов**.

### GetString

```rsl
GetString ( id [, prompt, len [, hide] ] )
```

Присваивает переменной типа `V_STRING` введённое значение. Ширина поля по умолчанию: **70 символов**.

### GetStringR

```rsl
GetStringR ( id [, prompt, len [, hide ] ] )
```

Аналогично `GetString`, но выравнивает введённое значение по правому краю. Ширина поля по умолчанию: **9 символов**.

### GetDate

```rsl
GetDate ( id [, prompt [, hide ] ] )
```

Присваивает переменной типа `V_DATE` введённое значение. Ширина поля: **10 символов**.

### GetTRUE

```rsl
GetTRUE ( id [, prompt ] )
```

Присваивает переменной типа `V_BOOL` введённое значение. Если пользователь ответил утвердительно — возвращает `TRUE`, переменная `id` получает `TRUE`.

```rsl
if (GetTRUE (TRUE, "Хотите напечатать отчет?"))
    OutReport;
end;
```

### Пример использования процедур ввода

```rsl
aa = "Иван";    /* Значение по умолчанию */
if (GetString (aa, "Как вас зовут?"))
    println ("Вас зовут: ", aa);
end;
/* На экране: "Как вас зовут? Иван" */
```

## Процедуры вывода

### Print

```rsl
Print ( ... )
```

Выводит в отформатированном виде значения переменных. Количество параметров не ограничено. Могут быть указаны спецификаторы форматирования.

### PrintLn

```rsl
PrintLn ( ... )
```

Аналогично `Print`, но после вывода выполняется возврат каретки.

### Message

```rsl
Message ( ... )
```

Выводит в нижнюю строку экрана сформированную отформатированную строку. Принимает произвольное количество параметров.

### SetOutput

```rsl
SetOutput ( [string] [, bool] )
```

Устанавливает новый файл для вывода.

| Параметр `bool` | Действие |
|-----------------|----------|
| Отсутствует или `FALSE` | Все данные из файла удаляются |
| `TRUE` | Информация дописывается в конец файла |

```rsl
oldFile = SetOutput ("My.out");
[ Этот текст выводится в файл My.out ];
SetOutput (NULL, TRUE);
[ Этот текст выводится в стандартный файл вывода ];
```

**Возвращает:** имя файла, в который выполнялся вывод до вызова.

### SetColumn

```rsl
SetColumn ( integer )
```

Формирует отчёт с переменным числом колонок. Устанавливает относительный номер колонки для вывода.

### FlushColumn

```rsl
FlushColumn
```

Выводит информацию из сформированного отчета в файл вывода.

### ClearColumn

```rsl
ClearColumn
```

Удаляет все данные из буфера печати.

### SetDefPrec

```rsl
SetDefPrec ( integer )
```

Устанавливает количество отображаемых по умолчанию знаков после точки для `V_DOUBLE` и `V_DOUBLEL`.

### SetOutHandler

```rsl
SetOutHandler ( NameMacro )
```

Устанавливает обработчик стандартного вывода. Для сброса — вызвать без параметров.

### GetPRNInfo

```rsl
GetPRNInfo ( [ escSeq: string, banner: string ] ) : string
```

Возвращает имя используемого устройства печати. В `escSeq` возвращается ESC-последовательность, в `banner` — файл заголовка.

### SetPRNInfo

```rsl
SetPRNInfo ( prnName: string [, escSeq: string, banner: string ] )
```

Устанавливает имя принтера, ESC-последовательность и файл заголовка.

## Типы и значения переменных

Процедуры преобразования типов. Если преобразование невозможно — возвращается нулевое значение соответствующего типа.

### ValType

```rsl
ValType ( val )
```

Возвращает тип переменной:

| Возвращаемое значение | Тип |
|-----------------------|-----|
| `V_UNDEF` | Неопределённый |
| `V_INTEGER` | Целое число |
| `V_MONEY` | Денежный |
| `V_DOUBLE` | Двойной точности |
| `V_DOUBLEL` | Расширенный двойной |
| `V_STRING` | Строка |
| `V_BOOL` | Логический |
| `V_DATE` | Дата |
| `V_TIME` | Время |
| `V_DTTM` | Дата-время |
| `V_FILE` | Файл |
| `V_STRUC` | Структура |
| `V_ARRAY` | Массив |
| `V_TXTFILE` | Текстовый файл |
| `V_DBFFILE` | DBF-файл |
| `V_PROC` | Процедура |

```rsl
if (ValType (MyVar) == V_UNDEF)
    MsgBox ("Значение переменной MyVar не определено")
end
```

### Double / DoubleL

```rsl
Double ( val )
DoubleL ( val )
```

Преобразует значение в тип `V_DOUBLE` / `V_DOUBLEL`.

### Int

```rsl
Int ( val )
```

Преобразует в тип `V_INTEGER`. Дата → количество дней от начала эры. Время → сотые доли секунды.

### String

```rsl
String ( val {, val} )
```

Принимает произвольное количество параметров и возвращает отформатированную строку.

### MkStr

```rsl
MkStr ( val1, val2 )
```

Возвращает строку заданной длины `val2`, состоящую из символа-заполнителя `val1`.

### Money

```rsl
Money ( val )
```

Преобразует в тип `V_MONEY`.

### Floor

```rsl
Floor ( val )
```

Возвращает наибольшее целое, меньшее или равное заданному. Для `V_DOUBLE`, `V_DOUBLEL`, `V_MONEY`.

### Asize

```rsl
Asize ( val [, newsize] )
```

Возвращает текущий размер массива `val`. Если указан `newsize` — удаляет старый массив и создаёт новый заданного размера.

### Date

```rsl
Date [ ( day, mon, year ) ]
```

Возвращает переменную типа `V_DATE`. Параметры заменяют соответствующие части текущей даты. Без параметров — возвращает текущую дату.

```rsl
aa = date;                    /* Текущая дата */
aa = date (10);               /* 10 число текущего месяца и года */
aa = date (10, 3);            /* 10 число 3 месяца текущего года */
aa = date ("", 3, 1994);      /* Текущий день марта 1994 */
```

### DateSplit

```rsl
DateSplit ( date, day, mon, year )
```

Разбивает дату на компоненты. Пропущенные компоненты заменяются нулевыми значениями.

```rsl
datesplit (date, NULL, mon);
```

### Time

```rsl
Time [ ( hour, min, sec, msec ) ]
```

Возвращает переменную типа `V_TIME`. Без параметров — текущее время.

### TimeSplit

```rsl
TimeSplit ( time, hour, min, sec )
```

Разбивает время на часы, минуты, секунды.

### DtTm

```rsl
DtTm ( date, time )
```

Создаёт значение типа `V_DTTM` из даты и времени.

### DtTmSplit

```rsl
DtTmSplit ( d, dt, tm )
```

Разбивает `V_DTTM` на дату и время.

### RubToStr

```rsl
RubToStr ( money [, rub, kop [, full ] ] )
```

Возвращает сумму прописью. Если `full` = `TRUE` — рубли и копейки в несокращённом виде.

```rsl
println (RubToStr (a * 100.00, b, c));       /* "Сто двадцать один руб. 11 коп." */
println (RubToStr (a * 100.00, b, c, TRUE));  /* "Сто двадцать один рубль 11 копеек" */
```

### RubToStrAlt

```rsl
RubToStrAlt ( money [, rub, kop ] )
```

Аналогично `RubToStr`, но с учётом падежа: рублей (рубля), копеек (копейки).

### CurToStrAlt

```rsl
CurToStrAlt ( money [, rub, kop ], ISO )
```

Возвращает сумму прописью в указанной валюте (ISO-код). Требуется файл `locale.ini`.

### MonName

```rsl
MonName ( mon )
```

Возвращает название месяца по номеру (1-12).

### NumToStr

```rsl
NumToStr (val, n1:string, n2:string, n3:string, isMan:bool, prec:integer) : string
```

Возвращает число `val` прописью. `n1`, `n2`, `n3` — три формы существительного. `isMan` = `TRUE` — мужской род. `prec` — знаков после точки (0-5, по умолчанию 2).

```rsl
println (NumToStr (125.45, "болт", "болта", "болтов", true, 3));
/* Выводит: сто двадцать пять целых сорок пять сотых болта */
```

## Работа со строками

Первый символ строки находится в позиции **1**.

### StrLen

```rsl
StrLen ( string )
```

Возвращает длину строки.

### Index

```rsl
Index ( string1, string2 )
```

Возвращает позицию `string2` в `string1` или **0**, если не найдена.

### StrBrk

```rsl
StrBrk ( string1, string2 )
```

Возвращает позицию первого символа из `string1`, который содержится в `string2`. Если не найден — **0**.

### SubStr

```rsl
SubStr ( string, integer1 [, integer2] )
```

Возвращает подстроку, начиная с позиции `integer1`. Если `integer2` указан — длиной `integer2`, иначе до конца строки.

### StrSet

```rsl
StrSet ( string1, integer, string2 )
```

Записывает `string2` в `string1`, начиная с позиции `integer`.

### Trim

```rsl
Trim ( string )
```

Удаляет начальные и конечные пробелы.

### StrSplit / StrSplit2

```rsl
StrSplit ( string, array, len [, len_first] [, minseg] )
StrSplit2 ( string, array, len [, len_first] [, minseg] )
```

Разбивает строку на сегменты длиной `len`. `StrSplit2` возвращает `TArray`.

### StrUpr / StrLwr

```rsl
StrUpr ( string [, len] )
StrLwr ( string [, len] )
```

Преобразуют строку в верхний / нижний регистр. `len` — количество символов для преобразования.

```rsl
TestLine = "Эта Строка Набрана В Разных Регистрах";
println ("В верхнем регистре – ", StrUpr (TestLine));
println ("В нижнем регистре – ", StrLwr (TestLine));
```

### CodeFor / StrFor

```rsl
CodeFor ( string )   /* ASCII-код первого символа */
StrFor ( number )    /* Символ по ASCII-коду */
```

### StrSubst

```rsl
StrSubst ( source, strToFind, strToReplace )
```

Заменяет в `source` все вхождения `strToFind` на `strToReplace`.

```rsl
Str = StrSubst (oldStr, "\n", " ");
```

### ToOEM / ToANSI

```rsl
ToOEM (string)    /* ANSI → OEM */
ToANSI (string)   /* OEM → ANSI */
```

## Параметры процедур

Первый параметр имеет номер **0**.

**Важно:** В методе класса первым параметром (№ 0) неявно передаётся `this`. Первый явный параметр метода имеет номер **1**.

### GetParm

```rsl
GetParm ( num, var )
```

Присваивает `var` значение фактического параметра вызывающей процедуры с номером `num`. Возвращает `TRUE` при успехе, `FALSE` при неудаче (var получает `V_UNDEF`).

### SetParm

```rsl
SetParm ( num, expression )
```

Присваивает фактическому параметру вызывающей процедуры с номером `num` значение `expression`. Возвращает `TRUE` при успехе.

## Математические процедуры

| Процедура | Описание |
|-----------|----------|
| `Exp ( number )` | Экспонента |
| `Log ( number )` | Натуральный логарифм |
| `Log10 ( number )` | Десятичный логарифм |

**Возвращаемое значение:** `V_DOUBLE` (или `V_DOUBLEL`, если параметр был `V_DOUBLEL`).

## Внешние программы

### Run

```rsl
Run ( prog, parm, init, finish )
```

Запускает внешнюю программу `prog` с командной строкой `parm`.

| Параметр | Описание |
|----------|----------|
| `init` | Сообщение при запуске. `">"` — перенаправить вывод в стандартный вывод RSL. `">><файл>"` — дозапись в файл |
| `finish` | Сообщение после завершения. Если задан — пауза перед возвратом |

```rsl
run (COMSPEC, "/c dir *.*", ">");
run ("myprog.com", NULL, NULL, "Нажмите любую клавишу");
run (COMSPEC, "/c " + "my.bat");
```

**Возвращает:** код возврата внешней программы или **-1**, если программа не найдена.

## Удалённый запуск макропрограмм

### CallRemoteRsl

```rsl
CallRemoteRsl (fileName [, procName [, parm1, parm2, ...]])
```

Запускает RSL-программу на терминале из серверного макроса. Требуется DLM-модуль `rsexts.d32`.

| Параметр | Описание |
|----------|----------|
| `fileName` | Имя макрофайла на терминале |
| `procName` | Имя вызываемой процедуры (необязательно) |
| `parm1, parm2` | Параметры процедуры (простые типы, не объекты) |

**Возвращает:** значение, возвращаемое `procName`.

```rsl
/* Макрофайл на сервере: */
import rsexts;
println (CallRemoteRsl ("remote.mac", "Main", $1.33L, 200));

/* Макрофайл на терминале (remote.mac): */
MsgBox ("This is remote.mac");
macro Main (p1, p2)
    MsgBox ("Macro Main called with ", p1, " ", p2);
    return "Main Return value"
end;
```

## Файлы и структуры

Процедуры работают с:
- Btrieve-файлами
- Текстовыми файлами
- Файлами формата DBF

Первый параметр — ссылка на файл/структуру (`V_FILE` или `V_RECORD`). Возвращают `V_BOOL`: `TRUE` при успехе, `FALSE` при ошибке.

### Термины

- **Текущая запись** — запись, прочитанная и находящаяся в буфере. Может отсутствовать при первом обращении, после `rewind`, при сбое.
- **Физическая позиция** — адрес записи в файле.

### ExistFile

```rsl
ExistFile ( string [, integer] )
```

Возвращает `TRUE`, если файл существует. Параметр `integer`:

| Значение | Где искать |
|----------|------------|
| 0 | Каталог текстовых файлов |
| 1 | Каталог Btrieve-файлов |
| 2 | Каталог DBF-файлов |
| Не задан | Текущий каталог |

### Open

```rsl
Open ( id [, filename] [, flag] [, bool] )
```

Открывает файл. Файл открывается автоматически при первом обращении, явный вызов необязателен.

| Параметр | Описание |
|----------|----------|
| `filename` | Имя файла, отличное от имени в словаре. `NULL` — использовать имя из словаря |
| `flag` | `1` — режим ускоренного доступа (только временные файлы!) |
| `bool` | `TRUE` — аварийное завершение при невозможности открыть; `FALSE` — вернуть `FALSE` |

```rsl
File aa (balance);
File bb (balance);
File cc () txt;
open (aa, "c:\\data\\bal.dbt");
open (bb, "c:\\arch\\balarch.dbt");
name = datadir + "data.txt";
open (cc, name);
```

### Close

```rsl
Close ( id )
```

Закрывает файл. Файл закрывается автоматически при выходе из контекста.

### ClearStructs

```rsl
ClearStructs
```

Освобождает из памяти структуры всех открытых Btrieve-файлов.

### Create

```rsl
Create ( id [, filename] [, bool] )
```

Создаёт новый Btrieve или DBF-файл по описанию из словаря. Не работает с текстовыми файлами (текстовые создаются автоматически).

### Clone

```rsl
Clone ( id [, filename] [, bool] )
```

Создаёт новый файл или удаляет информацию в существующем. Параметры аналогичны `Create`.

```rsl
File aa ("data.dbf") dbf write;
clone (aa);                        /* Удаляет информацию */
clone (aa, "newdata.dbf");         /* Создаёт новый файл */
```

### ViewFile

```rsl
ViewFile ( id [, edit] )
```

Просмотр текстовых и Btrieve-файлов. `edit = TRUE` — режим корректировки для Btrieve.

### DelFile

```rsl
DelFile ( string )
```

Удаляет файл.

### Next

```rsl
Next ( id [, integer] )
```

Считывает следующую запись (или первую, если текущей нет). Для текстовых файлов — читает следующую строку. Если `integer` указан — считает заданное количество байт.

### Prev

```rsl
Prev ( id )
```

Считывает предыдущую запись (или последнюю). Не работает с текстовыми файлами.

### ReWind

```rsl
ReWind ( id )
```

Переустанавливает файл. После этого `next` извлечёт первую запись, `prev` — последнюю.

### Insert

```rsl
Insert ( id [, string | size] [, integer] [, bool] )
```

Помещает новую запись в файл из буфера данных.

| Параметр | Описание |
|----------|----------|
| `string` | Для текстовых файлов — строка для записи |
| `size` | Для Btrieve — размер переменной части записи |
| `integer` | Количество байт для записи (без добавления символов конца строки) |
| `bool` | `TRUE` — аварийное завершение при дублировании ключа; `FALSE` — вернуть `FALSE` |

### Update

```rsl
Update ( id [, size] [, bool] )
```

Обновляет текущую запись. Не работает с текстовыми файлами.

### Delete

```rsl
Delete ( id, bool )
```

Удаляет текущую запись. Не работает с текстовыми файлами.

### WriteBlob / ReadBlob

```rsl
WriteBlob ( file, value )
ReadBlob ( file, value )
```

Работа с полями типа BLOB. `ReadBlob` возвращает `FALSE`, когда информация прочитана до конца.

```rsl
File ff (proba) write Blob;
File tt ("myfile") txt;

if (next (ff))
    WriteBlob (ff, "--------------------------------- Blob Data ---------------------------");
    WriteBlob (ff, date);
    WriteBlob (ff, time);
    while (next (tt))
        WriteBlob (ff, tt.str);
    end;
end;

rewind (ff);
if (next (ff))
    while (ReadBlob (ff, val))
        println (val);
    end;
end;
```

### GetPos

```rsl
GetPos ( id )
```

Возвращает физическую позицию текущей записи. Для DBF — порядковый номер записи. Не работает с текстовыми файлами.

### GetDirect

```rsl
GetDirect ( id [, recpos] )
```

Извлекает запись из физической позиции `recpos`. Если `recpos` не указан — используется позиция от последнего `getpos`.

```rsl
File ff (account);
getpos (ff);        /* Запоминает позицию */
getdirect (ff);     /* Извлекает запись */
```

### Поиск по ключу (GetEQ, GetGT, GetGE, GetLT, GetLE)

```rsl
GetEQ (id)   /* Равно */
GetGT (id)   /* Больше */
GetGE (id)   /* Больше или равно */
GetLT (id)   /* Меньше */
GetLE (id)   /* Меньше или равно */
```

Только для Btrieve-файлов. Возвращают `FALSE`, если запись не найдена. Значение ключа задаётся присвоением полям сегментов текущего ключа.

```rsl
FILE Счета (account) key 0;
Счета.Date = date;
Счета.Account = "010234";
getEQ (Счета);      /* Получена запись с заданным ключом */
```

### SetRecordAddr

```rsl
SetRecordAddr ( recId, fileId [, ind] [, offset] [, fix] )
```

Накладывает структуру `recId` на файл `fileId`. Позволяет работать с переменной частью записи и использовать структуры-union.

Смещение вычисляется: `GetRecordSize(recId) * ind + offset`

| Параметр `fix` | Действие |
|---------------|----------|
| `FALSE` или не задан | Смещение относительно начала переменной части |
| `TRUE` | Смещение относительно начала постоянной части |

```rsl
file fl (varlen) write;
record rec (vrec);

macro ShowVarLen (n)
    i = 0;
    while (i < n)
        SetRecordAddr (rec, fl, i);
        println (rec.id, " ", rec.name);
        i = i + 1;
    end;
end;

while (next (fl))
    ShowVarLen (file.nRec);
end;
```

### GetVarSize

### FileName

### NRecords

Возвращает количество записей в файле (не для текстовых).

### FldNumber / FldName / FldIndex / FldOffset

```rsl
FldNumber ( id )              /* Количество полей */
FldName ( id, number )        /* Имя поля по индексу */
FldIndex ( id, string )       /* Индекс поля по имени */
FldOffset ( id, string | number )  /* Смещение поля в байтах */
```

Не работают с текстовыми файлами. `FldOffset` — только Btrieve.

```rsl
File ff (account);

while (I < FldNumber (ff))
    println (I, " – ", FldOffset (ff, I));
end
```

### Copy


Копирует содержимое буфера `id2` в `id1`. Поддерживает `TBFile` и `TRecHandler`. При различной структуре выполняет поиск полей по имени с преобразованием типов.

### SetDelim

### Status

```rsl

```

Возвращает код ошибки Btrieve. Если `parm` задан — присваивает строку с сообщением об ошибке.

### ProcessTrn

```rsl
ProcessTrn ( TrnType [, MacroName, file1, file2, ...] )
```

Выполняет транзакцию с Btrieve-файлами.

| `TrnType` | Тип транзакции |
|-----------|---------------|
| 0 или `NULL` | Ничего не выполнять в вызывающем модуле |
| 1 | Рублевая проводка |
| 2 | Валютная проводка |
| 3 | Проводка валюты без рублей |
| 4 | Проводка вкладчиков |
| 5 | Ввод документов, счетов, клиентов, банков |
| 6 | Рублевая проводка "как есть" |
| 7 | Валютная проводка "как есть" |
| 8 | Проводка валюты без рублей "как есть" |
| 10 | Ввод параметров сделок |

**Возвращает:** `TRUE` — успех, `FALSE` — откат (`AbortTrn` или отрицательный ответ пользователя).

**Вложенные транзакции не поддерживаются.**

### ProcessConTrn

```rsl
ProcessConTrn ( TrnType [, MacroName, file1, file2, ...] )
```

Аналогично `ProcessTrn`, но выполняет конкурентную транзакцию. Требует `DSBLCNTRN=0` в `rsreq.ini`.

### LoopInTrn

```rsl
LoopInTrn ( val )
```

Если `val = TRUE` — транзакция прерывается и повторяется при захвате файлов. Количество повторов — `REPCOUNT`, задержка — `SLEEPTIME + RANDOMTIME` из `rsreq.ini`.

### AbortTrn

```rsl
AbortTrn ()
```

Завершает выполнение транзакции и производит откат всех изменений. `ProcessTrn` возвращает `FALSE`.

### SelectFile

```rsl
SelectFile ( file [, mask, head] )
```

Выводит диалоговое окно выбора файла. Возвращает `TRUE`, если файл выбран.

| Параметр | Описание | По умолчанию |
|----------|----------|-------------|
| `mask` | Шаблон имени файла | Текущий каталог |
| `head` | Заголовок окна | `"Выбор файла"` |

## Управление файлами и каталогами

### CopyFile

```rsl
CopyFile (src:string, dst:string [, ind:bool]) : bool
```

Копирует файл `src` в `dst`. `ind = TRUE` — отображать индикатор прогресса.

```rsl
if (not CopyFile ("cvt.c", "$rslDir\\cvt.c"))
    println ("Error Copy file");
end;
```

### RenameFile

```rsl
RenameFile (src:string, dst:string) : bool
```

Переименовывает файл. Файл назначения должен быть в том же каталоге.

### RemoveFile

```rsl
RemoveFile (src:string) : bool
```

Удаляет файл.

### MakeDir

```rsl
MakeDir (name:string) : bool
```

Создаёт каталог.

```rsl
if (not MakeDir ("$rslDir"))
    println ("Error creating dir");
end;
```
