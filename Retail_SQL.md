## Пакет `RSU_RTFM`

содержит процедуры, используемые для связи с подсистемой
"Финансовый мониторинг".
Процедуры

### `FM_Select`

```sql
(p_Branch:Number, p_PeriodFirstDate:Date, p_PeriodEndDate:Date, p_FlagCur:Number)
```

Процедура выполняет отбор операций для контроля в подсистеме "Финансовый
мониторинг".

**Параметры:**

шаблонов.
p_context – строка, содержащая контекст.
Возвращаемое значение:
Функция возвращает одно из значений:
· 1 – при успешном выполнении функции;
· 0 – в случае возникновения ошибки.
