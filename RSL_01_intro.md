# Введение

> RSL (R-Style Language) — Руководство программиста
> Copyright © 1993-2001 R-Style Softlab

---

Настоящее Руководство содержит описание языка RSL, являющегося неотъемлемой частью систем, разработанных программистами компании R-Style Softlab.

Руководство состоит из трёх глав, описывающих три основные части RSL:

| Часть | Описание |
|-------|----------|
| **Язык интерпретатора RSL** | Язык, при помощи которого пользователь имеет возможность создавать собственные программы |
| **Интегрированная среда разработки** | Средство для работы с Btrieve-файлами в интерактивном режиме, создания и редактирования диалоговых окон |
| **Отладчик макропрограмм** | Средство, позволяющее эффективно отлаживать RSL-программы |

RSL — современный язык, разработанный с учётом нынешних требований к программному обеспечению. В компании R-Style Softlab ведётся постоянная работа по его развитию и совершенствованию.
