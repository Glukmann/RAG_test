## Символьные константы

Под константой понимается значение одного из типов данных.

Символьные константы имеют тип `V_STRING` (максимальная длина — 256 символов) и задаются в виде последовательности символов, заключённых в кавычки:

```rsl
"а"
"F"
"абв_12"
"\n"
"\t"
```

## Управляющие последовательности

Последовательности, начинающиеся с обратной косой черты, аналогичны таковым в языке C:

| Последовательность | Значение |
|--------------------|----------|
| `\n` | Новая строка |
| `\r` | Возврат каретки |
| `\t` | Символ табуляции |
| `\f` | Перевод формата |
| `\xHH`, `\XHH` | Символ, заданный шестнадцатеричным кодом `HH` |
| `\\` | Одиночный символ `\` |

## Предопределённые константы

| Константа | Тип | Значение |
|-----------|-----|----------|
| `TRUE` | `V_BOOL` | Истина |
| `FALSE` | `V_BOOL` | Ложь |
| `NULL` | `V_UNDEF` | Неопределённое значение |

Предопределённые константы для работы с диалоговыми окнами описаны в разделе, посвящённом вводу/выводу.
