## Пакет `RSB_ACCLAIM`

содержит функции, используемые в механизме претензий к счетам и
для определения свободного остатка счета.
Функции

### `GetPriorityArrestClaim`

```sql
(p_Account:Varchar2, p_Chapter:Number, p_FIID:Number, p_BankDate:Date, p_Priority:Number):Integer Функция предназначена для получения минимального приоритета действующей претензии частичного ареста по счету на указанную дату.
```

**Параметры:**

p_Account – номер счета.
p_Chapter – глава счета.
p_FIID – валюта счета.
p_BankDate – дата, на которую определяется сумма.
p_Priority – приоритет.

**Возвращаемое значение:**

Функция возвращает минимальный приоритет действующей претензии частичного ареста.

### `GetSumArrestClaim`

```sql
(p_Account:Varchar2, p_Chapter:Number, p_FIID:Number, p_BankDate:Date, p_Priority:Number, p_Sum:Number):Integer Функция предназначена для получения суммы всех активных претензий ареста по счету на указанную дату.
```

**Параметры:**

p_Account – номер счета.
p_Chapter – глава счета.
p_FIID – валюта счета.
p_BankDate – дата, на которую определяется сумма.
p_Priority – приоритет.
p_Sum – сумма активных претензий.

**Возвращаемое значение:**

Функция возвращает сумму всех активных претензий ареста по счету.

### `GetSumSpecialClaim`

```sql
(p_Account:Varchar2, p_Chapter:Number, p_FIID:Number, p_BankDate:Date, p_Sum:Number):Integer Функция предназначена для получения суммы всех активных претензий целевого использования по счету на указанную дату.
```

**Параметры:**

p_Account – номер счета.
p_Chapter – глава счета.
p_FIID – валюта счета.
p_BankDate – дата, на которую определяется сумма.
p_Sum – сумма активных претензий.

**Возвращаемое значение:**

Функция возвращает сумму всех активных претензий целевого использования по счету.

### `IsAccountFullyArrested`

```sql
(p_Account:Varchar2, p_Chapter:Number, p_FIID:Number, p_BankDate:Date):Integer Функция предназначена для проверки наличия полного ареста по счету на указанную дату.
```

**Параметры:**

p_Account – номер счета.
p_Chapter – глава счета.
p_FIID – валюта счета.
p_BankDate – дата, на которую определяется сумма.

**Возвращаемое значение:**

Функция возвращает одно из значений:
- 1 – полный арест;
- 0 – в противном случае.
