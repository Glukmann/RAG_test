# RCE32

## Введение

Настоящее руководство содержит описание редактора RCE32.exe, предназначенного для
визуального создания ресурсов ИБС RS-Bank V.6.
Работа в редакторе осуществляется с помощью команд, объединенных в функциональные
группы согласно пунктам меню
.
Редактор ресурсов предоставляет следующие возможности:
- создание и редактирование ресурса
;
- загрузка и выгрузка ресурсов в файлы XML формата
.
Ресурсы типа PANEL, созданные в редакторе RCE32.exe, используются в качестве
источников данных в процедурах языка RSL при создании пользовательского интерфейса.
Подробное описание процедур языка RSL предоставлено в Руководстве "Проблемно-
ориентированный язык RSL" (файл Books\Tools\BnRSL.pdf).
При ссылке на информацию, расположенную в настоящем руководстве, указывается номер
страницы (в скругленном прямоугольнике 
), соответствующий нужному Вам разделу, или
гиперссылка. Ссылка на раздел другого руководства дается с указанием названия главы и
раздела, содержащего требуемую информацию.
Редактор ресурсов RCE32.exe
Редактор RCE32.exe предназначен для визуального создания ресурсов программы и
поставляется пользователям вместе с дистрибутивом.
Редактор позволяет создавать ресурсы следующих типов:
- PANEL – диалоговая панель.
- BSCROL – скроллинг по источнику данных с возможностью реализации стандартных
действий над данными при помощи инструмента.
- LSCROL – скроллинг по области памяти с поддержкой удаления, вставки строк при
помощи инструмента.
- SCROL – скроллинг по области памяти без возможности удаления, вставки строк при
помощи инструмента.
При создании собственного интерфейса пользователь может использовать ресурсы только
типа PANEL. Эта возможность реализуется при помощи процедуры RunDialog (см
Руководство "Проблемно-ориентированный язык RSL" глава "Встроенные процедуры"
раздел "Обработка диалоговых окон"). Процедура позволяет создавать диалоговую панель,
используя в качестве источника данных ресурсы типа PANEL, созданные в редакторе
RCE32.exe
. 
Ресурсы 
остальных 
типов 
нельзя 
использовать 
при 
создании
пользовательского интерфейса.
Ресурсы, созданные пользователем в редакторе RCE32.exe, сохраняются в библиотеке
ресурсов, которая представляет собой файл с расширением *.lbr. Все ресурсы, находящиеся
в библиотеке, идентифицируются по типу и имени. Файлы XML формата являются более
наглядными, поэтому в системе ресурсы хранятся в XML-файлах. В связи с этим редактор
ресурсов RCE32.exe поддерживает загрузку и выгрузку ресурсов в файлы XML формата
.

## Меню редактора ресурсов

Работа с ресурсами и библиотекой ресурсов ведется в следующих меню:
Примечание.
Меню Source и Menu являются устаревшими.
- Lib
- предназначено для работы с библиотекой ресурсов;
- Panel
- предназначено для создания и редактирования ресурсов;
- New
- предназначено для добавления на имеющийся ресурс новых элементов (рамки,
поля и текста);
- Item
- предназначено для редактирования элементов ресурса.
Lib
Меню содержит команды, используемые для работы с библиотекой ресурсов:
Примечание.
Команды Panel и Report являются устаревшими и не используются при работе
в редакторе.
- Open Lib – открытие библиотеки ресурсов.
- New Lib – создание новой библиотеки ресурсов.
- Find in Lib – поиск текстовой строки в библиотеке ресурсов.
- Get From XML file – импорт ресурсов из заданного XML-файла в библиотеку ресурсов.
- Get From Directory – импорт ресурсов из заданного каталога, содержащего XML файлы, в
библиотеку ресурсов.
- Export To XML file – экспорт всех ресурсов библиотеки в один заданный XML-файл.
- Export To Directory – экспорт всех ресурсов библиотеки в заданный каталог.
- Close Lib – закрытие библиотеки ресурсов.
- Compress Lib – сжатие библиотеки ресурсов.
- Check Lib – проверка всех ресурсов библиотеки на наличие ошибок и корректность
отображения на экране. Все ресурсы библиотеки поочередно отображаются на экране.
Переход к следующему ресурсу осуществляется по клавише [Esc].
- Check fast – быстрая проверка всех ресурсов библиотеки. На экране отображаются
только ошибки, найденные в ресурсах.
- Print Lib – печать библиотеки ресурсов.
- Save as … – сохранение библиотеки ресурсов.
- Change dir – изменение каталога, используемого редактором RCE32.exe по умолчанию.
При запуске редактора в качестве каталога по умолчанию используется каталог ..\Obj.
- DOS shell – временный переход в режим командной строки MS-DOS.
- Quit – выход из редактора.

Команды Save panel Old и Define Scroll не используются при создании
пользовательского интерфейса.

В окне выбора ресурса для редактирования предусмотрены следующие действия:
[F6] – Сортировка ресурсов. Возможные варианты сортировки:

- по типу;
- по дате;
- по комментарию.
[F8] – Удаление ресурса из библиотеки ресурсов.
[Enter] – Выбор ресурса для редактирования.
[Esc] – Выход из окна выбора ресурса для редактирования.
- New panel – создание нового ресурса.
- Save panel – сохранение изменений в ресурсе.
- Save as… – сохранение ресурса под новым именем.
- Save as XML – экспорт заданного ресурса из библиотеки в XML-файл.
- Erase panel – удаление ресурса из загруженной библиотеки.
- Test panel – тестирование работы ресурса.
- Test panel EW – тестирование работы ресурса в графическом режиме.
- Show nums – отображение номеров полей в ресурсе. Выполнение действия возможно
также по клавише [F5].
- Show groups – отображение номера групп полей в ресурсе. Выполнение действия
возможно также по клавише [F6].
- Show names – отображение наименования полей в ресурсе (наименование задается в
параметре поля Field name
). Выполнение действия возможно также по клавише [F7].
- Contrast panel – контрастное отображение ресурса для выделения на нем всех объектов
(полей, текста и т.д.) для оценки их размеров и размещения. Выполнение действия
возможно также при помощи клавиш [Alt+F9].
- Information – информация о библиотеке ресурсов или ресурсе.
New
Меню содержит команды, предназначенные для добавления на имеющийся ресурс новых
элементов (рамки, поля и текста):
- Bar –добавление рамки на ресурс.
- Text – добавление текста на ресурс.
- Field – добавление поля на ресурс. Выполнение действия возможно также по
клавише [Ins].
Item
Меню содержит команды, предназначенные для редактирования элементов ресурса:
- Remove – удаление выбранного элемента из ресурса. Выполнение действия возможно
также по клавише [Del].
- Move – перемещение элемента, на котором стоит курсор. Выполнение действия возможно
также при помощи клавиш [Ctrl+F5]. Перемещение элемента осуществляется с помощью:
- клавиш управления курсором;
- мыши.
- Edit – редактирование элемента, на котором стоит курсор. Выполнение действия
возможно также при помощи клавиш [F4] или [Enter]. Если курсор не указывает ни на один
элемент, включается режим редактирования параметров ресурса.

## Создание ресурса

Перед началом работы по созданию ресурса в редакторе необходимо открыть уже
существующую библиотеку ресурсов, используя команду Lib\Open Lib, или создать новую
библиотеку ресурсов с помощью команды Lib\New Lib.
При выполнении команды Panel\New panel вызывается окно, в котором необходимо задать
имя создаваемого ресурса.
Примечание.
Изменять размер ресурса можно следующим образом: спозиционировать
курсор в правом нижнем углу ресурса и, удерживая клавишу мыши, изменить
размер ресурса на требуемый. Перед изменением размера ресурс меняет
цвет.
Редактор ресурсов позволяет устанавливать параметры ресурса
.
Редактор RCE32.exe дает возможность добавлять на созданный ресурс следующие
элементы:
- текст
;
- поле
;
- рамку
.
Работа по редактированию ресурса осуществляется при помощи команд меню Panel.
Примечание.
Редактировать элементы ресурса удобнее при помощи мыши:
- перемещать элементы (поля, тексты и т.д.) – спозиционировать курсор на
элементе и, удерживая  клавишу мыши, переместить элемент в требуемое
место;
- изменять размер элементов (длину полей) – спозиционировать курсор в
правом нижнем углу элемента и, удерживая клавишу мыши, изменить
размер элемента на требуемый.

Установка отметки в параметре осуществляется по клавише [Space].
- NO TABSTOP – признак исключения поля из стандартного маршрута обхода полей
ресурса по клавише [Tab]. Установка отметки в параметре осуществляется по
клавише [Space].
- ADD GROUPING – признак отображения числовых значений в поле с группировкой по
разрядам. Разряды отделяются друг от друга апострофами. Установка отметки в
параметре осуществляется по клавише [Space].
Если для ресурса установлен параметр NO AUTODIR
, то в нижней части справа
пользователь может задать порядок перемещения по ресурсу при помощи клавиш
управления курсором. Для этого в качестве значений параметра задаются номера полей
ресурса, в которые устанавливается курсор при нажатии клавиш управления курсором.
Добавление рамки на ресурс
Добавление рамки на ресурс осуществляется при помощи команды New\Bar, при этом в
специальном окне необходимо выбрать стиль рамки:
- отсутствует – границы рамки отсутствуют;
- одинарная – все границы рамки оформляются одной чертой;
- двойная – все границы рамки оформляются двойной чертой;
- комбинированная 1 – верхняя и нижняя границы рамки оформляются двойной чертой,
боковые границы оформляются одной чертой;
- комбинированная 2 – верхняя и нижняя границы рамки оформляются одной чертой,
боковые границы оформляются двойной чертой;
- сплошная – границы рамок оформляются сплошной чертой.
Загрузка и выгрузка ресурсов в файлы
XML формата
Загрузка и выгрузка ресурсов в файлы XML формата выполняется при помощи следующих
команд редактора RCE32.exe:
- Lib\Export To XML file – все ресурсы библиотеки помещаются в один XML-файл.
- Lib\Export To Directory – для каждого ресурса создается отдельный XML-файл, при этом:
- если заданный каталог не существует, то программа автоматически создаёт этот
каталог;
- если при экспорте в каталоге уже существует XML-файл для экспортируемого ресурса,
то он автоматически перезаписывается.
- Panel\Save as XML – выгрузка отдельного ресурса библиотеки в XML-файл.
- Lib\Get From XML file – если импортируется ресурс, который уже существует в
библиотеке, то он автоматически перезаписывается.
- Lib\Get From Directory – если импортируется ресурс, который уже существует в
библиотеке, то он автоматически перезаписывается.
При импорте ресурсов из XML-файлов выполняется проверка XML-файла на соответствие
XSD-схеме. Файл со схемой reslib.xsd должен располагаться в том же каталоге, что и файл
исполняемой программы RCE32.exe.
