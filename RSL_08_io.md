## Конструкция WITH

Конструкция `WITH` применяется для обращения к конкретным экземплярам класса.

### Синтаксис

```rsl
WITH (идентификатор)
    <список инструкций>
END
```

- `идентификатор` — имя переменной, содержащей ссылку на объект RSL-класса.
- Список инструкций выполняется для указанного объекта.

### Пример

```rsl
With ( this )
    Println ("-----------------");
    Отчет;
    Println ("-----------------");
End;
```

В приведённом примере для объекта `this` выполняется метод `Отчет` и печатаются две пунктирные линии.

## Способы ввода и вывода

### Ввод данных

Ввод данных возможен двумя способами:

| Способ | Описание |
|--------|----------|
| **Интерактивный** | Ввод данных с клавиатуры |
| **Из файлов** | Btrieve-файлы, текстовые файлы и файлы формата DBF |

### Вывод данных

| Способ | Описание |
|--------|----------|
| **Стандартное устройство** | Файл, в который выводят процедуры `print`, `println` и инструкция вывода. Имя файла определяется вызывающей программой |
| **В файлы** | Текстовые файлы, Btrieve-файлы и DBF-файлы |

## Спецификаторы форматирования

Спецификаторы форматирования определяют вид, в котором данные выводятся процедурами `print`, `println` и инструкцией вывода. Эти спецификаторы позволяют форматировать как строку в памяти (при помощи процедуры `string`), так и данные, отображаемые процедурой `message`.

Список спецификаторов форматирования указывается сразу же после выражения, к которому он относится. Он начинается с двоеточия и не должен иметь пропусков. Спецификаторы в этом списке отделяются друг от друга двоеточиями.

### Числовые спецификаторы

| Спецификатор | Описание |
|--------------|----------|
| `ширина` | Ширина поля вывода. Если значение меньше заданной ширины, оно дополняется пробелами. Если ширина равна нулю или не указана — значение выводится полностью |
| `точность` | Количество цифр после десятичной точки для данных типа `V_DOUBLE`. Если равно нулю — дробная часть не выводится |

### Символьные спецификаторы

| Спецификатор | Описание |
|--------------|----------|
| `l` | Выравнивание по левому краю поля вывода (по умолчанию для строк) |
| `r` | Выравнивание по правому краю поля вывода (по умолчанию для чисел) |
| `c` | Выравнивание по центру поля вывода |
| `a` | При выводе чисел — применение апострофов для отделения тысяч |

### Пример

```rsl
print (a + b:10:5:c)
```

В данном примере для переменной `b` указаны следующие спецификаторы: ширина поля вывода 10 символов, вывод 5 знаков после запятой, печать значения по центру поля.
